This is a prototype for the graph database construction of metabolic pathway (nucleotide salvage pathway) that constains 
every bit of information downloaded from bigg website. 

1. Unzip the testdata and download the neo4j frontend. Choose the unzipped folder and load the database through neo4j.
2. Nevigate to the neo4j web interface to play around with the graph database by using their query language called Cypher.
3. All the input files are stored in the tables.zip.
